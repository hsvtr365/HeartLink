[GLOBAL UI RULES]
- 상단 타이틀(AppBar) 없음
- 하단 탭 아이콘만 네비게이션 역할
- 현재 위치는 하단 아이콘 활성 상태로만 표시
- 상세 화면은 스택 네비게이션(push)
- VN 씬 / 몰입 화면에서는 하단 탭 숨김 가능

────────────────────────────────

[BOTTOM TAB - ICON ONLY / 5 TABS]
1. 💬 캐릭터 대화창
2. ❤️ 추천 캐릭터
3. 🧩 전체 캐릭터 / 업데이트
4. 🏆 퀘스트 / 배지
5. 👤 마이페이지

앱 최초 진입 기본 탭: 💬 캐릭터 대화창

────────────────────────────────

[💬 캐릭터 대화창 - ROOT]
- 전체: 리스트 화면
- 리스트 아이템 구성
  - 캐릭터 썸네일
  - 캐릭터 이름
  - 진행 상태 텍스트 (짧게)
    - EP.2
    - NEW
    - DONE
  - 상태 아이콘
    - 새 콘텐츠 표시
    - 다운로드 필요 표시
- 빈 상태
  - 안내 일러스트
  - 짧은 문구
  - ❤️ 추천 캐릭터 탭 이동 유도

[💬 캐릭터 대화창 - DETAIL]
- 상단 타이틀 없음
- 말풍선 형태 대화 UI
- 유저 입력 없음, 선택지 버튼만 제공
- 에피소드 종료 시
  - 시스템 메시지
  - [만남으로 이동] 버튼 표시

────────────────────────────────

[❤️ 추천 캐릭터 - ROOT]
- 풀스크린 카드 스와이프 UI
- 카드 구성
  - 캐릭터 이미지
  - 캐릭터 이름
  - 태그 요약
- 하단 고정 버튼
  - ❌ 패스
  - ❤️ 좋아요
- ❤️ 좋아요 시
  - 캐릭터 슬롯 생성
  - 캐릭터 대화창 상세로 이동

────────────────────────────────

[🧩 전체 캐릭터 / 업데이트 - ROOT]
- 스크롤 가능한 카탈로그 화면
- 구성 순서
  1. 태그 기반 추천 캐릭터 (가로 스크롤)
  2. 전체 태그 목록 (칩 형태)
  3. 캐릭터 목록 (그리드 또는 리스트)
- 캐릭터 카드
  - 이미지
  - 이름
  - 상태 뱃지
    - PLAY
    - UPDATE
    - LOCK
- 캐릭터 선택 시
  - 캐릭터 정보 페이지 이동
  - Play 버튼 → 캐릭터 대화창 상세 이동

────────────────────────────────

[🏆 퀘스트 / 배지 - ROOT]
- 기본 진입 화면: 퀘스트
- 상단 타이틀 없음
- 내부 전환
  - 아이콘 기반 토글 또는 세그먼트
    - 퀘스트
    - 배지
- 퀘스트
  - 카드 리스트
  - 진행률 바
  - 보상 표시
- 배지
  - 그리드 형태
  - 잠김 / 획득 상태 시각적 구분

────────────────────────────────

[👤 마이페이지 - ROOT]
- 상단 프로필 영역
  - 닉네임
  - (옵션) 아바타
- 설정 리스트
  - 닉네임 변경
  - 캐릭터 커스텀
  - 계정 관리 (구글 / 애플)
  - 문의
  - 약관 / 앱 정보

────────────────────────────────

[VN SCENE - 만남 연출]
- 하단 탭 숨김
- 전체 화면
- 배경 + 캐릭터 일러스트
- 대사 창
- 선택지 버튼
- 종료 시
  - 💬 캐릭터 대화창 상세로 복귀
